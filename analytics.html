CHEF ANALYTICS

An intelligence platform that allows customers to query, visualize, and report on any operational events within their Chef-managed infrastructure.



Query

Search for and export events generated by Chef

Visualize

See the state of your infrastructure at a glance

Report

Export matching events to CSV or PDF

Notify

Send interesting events to other systems

Visual Timeline

See all your Chef runs and Chef server operations in one unified timeline that can be filtered and queried. See how nodes have changed over time.

Audit and Compliance Capabilities

Use Chef Client’s audit mode combined with custom rules to be immediately alerted when nodes fail compliance checks -- even if their converge succeeded.

Write Custom Rules

Use Analytics’ rule language to generate additional events that can be sent to external systems like email, Hipchat, webhook-based systems, or Splunk whenever any condition occurs.
